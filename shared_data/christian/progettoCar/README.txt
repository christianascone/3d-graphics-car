Compilazione
----------

Requisiti:
- freeglut3-dev

Per la compilazione si consiglia di portarsi all'interno della directory del progetto e di lanciare il comando:

make

La compilazione dovrebbe essere eseguita senza errori generando gli eseguibili.

Esecuzione
-----------

Mantenendosi all'interno della directory del progetto, per l'esecuzione dell'eseguibile, lanciare il seguente comando:

./progettoCar
